## 📐 **Responsive Breakpoints**

```css
Desktop:  >1200px - Full layout
Laptop:   992px-1200px - Slightly condensed
Tablet:   768px-992px - 2-column layouts → 1-column
Mobile:   480px-768px - Single column, larger text
Small:    <480px - Optimized for smallest phones
```
